# CharleBin

**CharleBin** est un outil de partage de textes sécurisé et éphémère.

## Ce que fait le projet
Ce projet permet de stocker et de partager des extraits de texte avec des options de contrôle avancées :
- **Sécurisation** : protection par mot de passe recommandée pour chaque partage.
- **Expiration** : paramétrage de la durée de vie du contenu (ex. : 1 mois).
- **Autodestruction** : option d'effacer le texte immédiatement après sa lecture.
- **Collaboration** : possibilité d'autoriser ou non les commentaires sur le contenu partagé.

## Prérequis
Les prérequis sont :
- `make`
- un navigateur web

Pour installer en local :
