# CharleBin

**CharleBin** est un outil de partage de texte sécurisé et éphémère.

## Ce que fait le projet
Ce projet permet de stocker et de partager des extraits de texte avec des options de contrôle avancées :
* **Sécurisation** : Protection par mot de passe recommandé pour chaque partage.
* **Expiration** : Paramétrage de la durée de vie du contenu (ex: 1 mois).
* **Autodestruction** : Option pour effacer le texte immédiatement après sa lecture.
* **Collaboration** : Possibilité d'autoriser ou non la discussion sur le contenu partagé.
